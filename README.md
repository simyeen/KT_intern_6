# 🤩 KT\_인턴\_6조 (Wake_up 지니!)

## 서비스 소개

- 졸음 운전을 카메라 인식으로 파악하고, 지니를 통해 운전자를 깨워준다.

## 기능 구현 목록

- 사용자의 모습 보여주기.
- 현재 GPS로 위치 보여주기.
- 졸음 감지시에 가까운 졸음센터 안내하기
